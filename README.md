# osu! Imagemap Creator

A simple web tool to generate imagemap BBCode for osu! profiles.

## Features

- Load images via URL
- Create interactive imagemaps
- Generate BBCode for osu! forums

## Usage

1. Open `https://slep34263657.github.io/osu-imagemap-creator` in your browser.
2. Paste an image URL in the sidebar.
3. Click "Load Image" to display it.
4. Use the tools to create your imagemap.
5. Copy the generated BBCode.
